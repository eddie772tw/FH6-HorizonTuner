Hotkeys

  • Lock / Unlock Ctrl+L
  • Save position Ctrl+S
  • Restore position Ctrl+Shift+R
  • Re-learn limiter Ctrl+R
  • Restart HUD Ctrl+Alt+R

Connection

In FH6 go to Settings → HUD and Gameplay and set:
Data Out: ON
IP Address: 127.0.0.1
Port: 5301

Turn off the in-game speedometer to avoid running two HUDs simultaneously.

About

For those who don't know, this is basically an evolution from what I did as the custom HUD for Forza Horizon 5.
This time it has more features, quality of life improvements and certain bug fixes that weren't entirely done for FH5's Custom hud, trying to make things as painless as possible for the end user as well.
Not gonna lie though, I still find it incredibly funny that this entire project was born out of a custom skin i made for GamepadViewer which ended up including a bit of HTML to have its own temeletry being displayed like one of those sim racers where it tracks the throttle and brake, etc.
Like the previous HUD, I hope you enjoy this one as much as it was fun for me to develop it.

If you want to use it while streaming on OBS, make sure to use these settings: Window Capture → Capture Method: Windows 10 (1903+)
